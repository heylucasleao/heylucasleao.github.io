# Fluxo de Decisão

A fim de facilitar para identificar a melhor abordagem causal, acabei criando um infográfico dinâmico. Basta responder a questões sobre o desenho do estudo e as premissas dos  dados, que ele mapeará o caminho e direcionamento para isolarmos o efeito de interesse. Ele não é uma bala de prata, mas é mais para ter um norte.
